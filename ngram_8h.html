◆ enigma_crack_plugboard_ngram()

void enigma_crack_plugboard_ngram ( enigma_crack_config_t *  config,
enigma_ngram_list_t *  ngramList 
)

Crack plugboard settings using n-gram scoring.

This function attempts to determine the plugboard settings used in the ciphertext's Enigma key by evaluating different plugboard combinations and scoring the resulting plaintext using n-gram frequencies.

Resulting scores and configurations are printed in descending order.

Parameters
configPointer to the cracking configuration structure.
ngramListPointer to the n-gram list structure.
