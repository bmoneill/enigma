Detailed Description

Represents a rotor configuration for the Enigma machine.

Indices here are entered manually to avoid extra calculation at runtime. Essentially, they map the rotor's alphabet to the standard alphabet (A-Z).

Parameters
nameName of the rotor.
fwd_indicesCharacter code of each standard alphabetic character in the rotor's alphabet.
rev_indicesCharacter code of each rotor's alphabetic character in the standard alphabet.
notchesArray of notch positions.
notches_countNumber of notches.
